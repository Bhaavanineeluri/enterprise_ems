# Enterprise EMS

Enterprise Management System built using FastAPI, SQLAlchemy, MySQL, Alembic and JWT authentication.

## Technology Stack

Backend:
- FastAPI
- Python
- SQLAlchemy ORM
- MySQL
- Alembic

Security:
- JWT Authentication
- Refresh Tokens
- OTP/MFA
- RBAC

Architecture:
- Modular Architecture
- Repository Pattern
- Service Layer
- Unit of Work
